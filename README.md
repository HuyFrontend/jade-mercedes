MBGo - Mercedes Template
====================
Client: rapp.com
## Requirements

> - NodeJS 
> - gulp, gulp-sass, gulp-jade

## Frontend Technical
> - Bootstrap 3.3.7 (sass verison)
> - jQuery 3.2.1
> - Jade
> - Sass

## Quickstart
``` 
$ npm i gulp -g
$ npm i
```
```bash
$ npm run build
$ npm run server
```
